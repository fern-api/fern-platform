The TypeScript Node Browser SDK generator outputs a fully functional TypeScript/JavaScript
SDK for usage in the browser. It can publish the SDK to npmjs.org (or any other npm repository).

Checkout the source code [here](https://github.com/fern-api/fern-typescript)

### Config options

- `namespaceExport`: Allows you to control the name of the generated namespace export and client class.

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-typescript-browser-sdk
  version: 0.7.2
  output:
    location: npm.buildwithfern.com
    package-name: "@fern-{your-org}/plantstore"
  config:
    namespaceExport: PlantStore
```
</CodeBlock>
